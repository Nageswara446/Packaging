# AVC-Cloud-RAPPS

# Description

This repo is used for automated RAPPS Cloud Image Build. The repo consists of code in PowerShell, Groovy, Ansible and Terraform to build, deploy the images to a Shared Gallery and then delivering them via Citrix MCS machine creation services.
